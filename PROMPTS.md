# 💡 Промпты для Claude Code — Реализация фич Монополии

## 🎯 Приоритет 1: Система тюрьмы

### Промпт:
```
Реализуй полную систему тюрьмы в game-full.jsx:

1. Игрок попадает в тюрьму в следующих случаях:
   - Земля на клетку "Карантин" (pos=30)
   - Выбросил 3 дубля подряд
   - Вытянул карточку события с требованием

2. В тюрьме игрок может выйти тремя способами:
   a) Заплатить 50 денег
   b) Использовать карточку "Бесплатно выходите из тюрьмы"
   c) Выбросить дубль (3 попытки на каждый ход)

3. Логика:
   - Проверь поле inJail в state.players[idx]
   - jailTurns считает отсиженные ходы
   - Если выбросил дубль в тюрьме → автоматический выход
   - После выхода игрок делает обычный ход с новыми кубиками

4. UI:
   - Показывай сообщение "Ты в тюрьме! Осталось X ходов"
   - Добавь кнопку "Выйти за 50"
   - При дубле автоматически освобождай

5. Тестируй:
   - Попадание на клетку 30
   - 3 дубля подряд
   - Выход по дублю
   - Выход по платежу
```

## 💳 Приоритет 3: Залог недвижимости

### Промпт:
```
Реализуй систему залога в game-full.jsx:

1. Условия залога:
   - Игрок владеет клетку
   - На клетке нет домов/отелей (сначала нужно снести)
   - Залог дает 50% от стоимости клетки

2. Логика:
   - mortgaged: { cellIdx: true } отслеживает залоги
   - mortgageProperty(cellIdx, playerId) залогирует
   - redeemProperty(cellIdx, playerId) выкупит (стоимость * 1.1)
   - За залоговую клетку аренду не платят

3. Когда нужен залог:
   - Автоматический залог при недостатке денег для оплаты аренды
   - Модальное окно: выбери клетки для залога (сортировка по стоимости)
   - Рекурсивно залогируй до нужной суммы

4. UI:
   - В PortfolioScreen помечай залоговые города как "⚠️ В залоге"
   - Кнопка "Выкупить" рядом с залоговой клеткой
   - Снять дома перед залогом (если есть)

5. Граф:
   - За залоговую клетку аренда = 0
   - Залог показывается на доске полупрозрачностью

6. Тестируй:
   - Успешный залог и выкуп
   - Попытка залога клетки с домами (должна быть ошибка)
   - Автоматический залог при недостатке денег
```

## 💀 Приоритет 5: Система банкротства

### Промпт:
```
Реализуй автоматическое банкротство игроков:

1. Условие банкротства:

2. Логика:

3. Конец игры:
   - Когда остаётся 1 игрок, он побеждает
   - Показать финальный экран с победителем
   - Статистика: сколько каких городов, сколько денег и т.д.

4. UI:
   - Помечай банкротов на LeaderboardScreen (красный фон)
   - Сообщение: "Игрок {name} разорился"
   - Финальный экран победы

5. Правила:
   - Если игрок не может заплатить аренду → автозалог имущества
   - После автозалога вновь проверь, хватает ли денег
   - Если всё равно не хватает → банкротство

6. Тестируй:
   - Попади на дорогую аренду без денег
   - Автозалог имущества
   - Финальный экран победы
```

## 🔍 Как использовать эти промпты

1. Откройте Claude Code
2. Загрузьте папку `claude-code/`
3. Возьмите промпт из нужного приоритета
4. Вставьте в Claude Code
5. Разработчик реализует фичу
6. Тестируйте в браузере

## ⚠️ Общие правила

- **Не удаляй существующий код** — дополняй
- **Сохраняй структуру state** — добавляй новые поля осторожно
- **Тестируй в игре** — каждая фича должна быть кликабельна
- **Документируй функции** — комментарии в коде
- **Баланс важен** — не делай фичу слишком мощной

## 📞 Если что-то не работает

1. Проверь консоль браузера (F12)
2. Убедись, что измененные функции экспортированы в `window`
3. Перезагрузи страницу (Ctrl+Shift+R)
4. Проверь, что state обновляется корректно
