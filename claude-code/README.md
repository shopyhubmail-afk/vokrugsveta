# 🎲 "Вокруг света" — Полнофункциональная игра Монополия

## 📋 Описание проекта

Телеграм mini app с полной реализацией классической игры Монополия. Игра включает все основные механики, интерактивные вкладки для анализа портфеля и рейтинг лидеров.

## 🎮 Основные механики

### Уже реализовано:
- ✅ Броски кубиков с анимацией
- ✅ Система дублей и дополнительных ходов
- ✅ Три дубля подряд → тюрьма
- ✅ Прохождение старта → +200 денег
- ✅ Покупка недвижимости
- ✅ Система аукционов при отказе
- ✅ Расчет аренды с учетом домов/отелей
- ✅ Монополии (сбор всех полей одного цвета = 2x аренда)
- ✅ События и карточки (Удача, События, События мира)
- ✅ Система налогов
- ✅ Логирование всех транзакций

## 🎯 Архитектура

### Стек технологий:
- **React 18** (без JSX в файлах, используется Babel standalone)
- **Vanilla JavaScript** для логики
- **CSS 3** (переменные, grid, flexbox)
- **Tabler Icons** для иконок

### Основной Flow:
1. `index.html` загружает все зависимости и JSX файлы через `<script type="text/babel">`
2. `board-data.js` инициализирует конфиг поля (REGIONS, CELLS)
3. `game-full.jsx` содержит хук `useGame()` со всей логикой
4. `app-tabs.jsx` оборачивает игру в систему вкладок
5. Компоненты рендерятся в `#root` элемент

### State структура:
```javascript
{
  players: [{ id, name, pos, balance, inJail, jailTurns, bailCards, bankrupt }],
  current: 0, // индекс текущего игрока
  dice: [1-6, 1-6],
  rolling: boolean,
  doubleCount: 0, // количество дублей подряд
  phase: 'idle' | 'rolling' | 'moving' | 'buy' | 'rent' | 'auction' | 'special' | 'jail',
  landed: null | cellIndex,
  owners: { cellIdx: playerId }, // кто владеет клеткой
  houses: { cellIdx: count }, // количество домов (5 = отель)
  mortgaged: { cellIdx: true }, // заложенные клетки
  logs: [{ type, player, amount, property, timestamp }], // история
  hint: string,
  toast: string,
}
```

## 🔧 Ключевые функции

### game-full.jsx
- `useGame(count)` — инициализирует игру на N игроков
- `rollDice()` — бросок кубиков с анимацией
- `stepMove(pIdx, steps)` — движение игрока на N клеток
- `land(pIdx)` — обработка приземления (покупка, аренда, события)
- `isMonopoly(region)` — проверка наличия монополии
- `calcRent(cell)` — расчет аренды с учетом домов

### Экраны (app-tabs.jsx)
- `GameScreen` — интерактивная доска + лента клеток
- `PortfolioScreen` — портфель игрока (баланс + активы)
- `LeaderboardScreen` — рейтинг по состоянию
- `LogbookScreen` — история всех транзакций
- `AnalyticsScreen` — статистика баланса и регионов
