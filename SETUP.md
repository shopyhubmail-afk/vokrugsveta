# 🚀 Инструкция по загрузке в Claude Code

## Шаг 1: Подготовка папки
Эта папка содержит полный проект игры "Вокруг света":

```
claude-code/
├── index.html              ← Главный файл приложения
├── README.md               ← Описание проекта
├── PROMPTS.md              ← Промпты для разработки
├── SETUP.md                ← Этот файл
└── vokrug-sveta/
    ├── styles.css          ← Все стили (переменные, layout)
    ├── board-data.js       ← Конфиг поля (40 клеток, 8 регионов)
    ├── Board.jsx           ← Компонент доски (3D, пешки, дома)
    ├── game-full.jsx       ← Основная логика игры
    ├── panels.jsx          ← Панели действий
    └── app-tabs.jsx        ← Вкладки и экраны анализа
```

## Шаг 2: Как запустить локально

### Вариант A: Простой HTTP сервер
```bash
cd claude-code
python3 -m http.server 8000
# или
npx http-server
```
Откройте `http://localhost:8000` в браузере

### Вариант B: Live Server (VS Code)
- Установи расширение "Live Server"
-右-click на `index.html` → "Open with Live Server"

## Шаг 3: Как использовать в Claude Code

1. **Откройте Claude Code** (claude.ai с режимом разработки)
2. **Загрузите папку** `claude-code/` в проект
3. **Откройте `PROMPTS.md`** — там 8 приоритетных задач
4. **Выберите фичу** из приоритетов (например, "Система тюрьмы")
5. **Скопируйте промпт** и вставьте в Claude Code
6. **Дайте команду**: "Реализуй эту фичу в game-full.jsx"

## Шаг 4: После каждого изменения

1. Claude Code сохраняет файлы автоматически
2. Обновите браузер (F5)
3. Протестируйте фичу в игре
4. Если ошибки → смотрите консоль браузера (F12)

## 🎮 Как тестировать

### Основной сценарий:
1. Откройте вкладку "Игра"
2. Нажмите "Бросить кубики"
3. Следуйте подсказкам в интерфейсе
4. Переходите между вкладками "Портфель", "Рейтинг", "Логбук", "Анализ"

### Быстрые тесты:
- **Покупка города**: Выпадите на свободный город → "Купить"
- **Аренда**: Выпадите на город противника → должна показаться сумма аренды
- **Дубли**: Выбросьте одинаковые числа → получите ещё ход
- **Таксы**: На клетку "Таможня" или "Сбор" → минус деньги
- **События**: На "Событие" или "Удача" → случайная карточка

## ⚙️ Структура игрового состояния

```javascript
{
  players: [
    { id, name, pos, balance, inJail, jailTurns, bailCards, bankrupt }
  ],
  current: 0,           // индекс текущего игрока
  dice: [1-6, 1-6],     // текущие кубики
  rolling: false,       // считаю ли кубики
  doubleCount: 0,       // количество дублей подряд
  phase: 'idle' | ...,  // текущая фаза хода
  landed: null,         // индекс последней клетки
  owners: {},           // { cellIdx: playerId }
  houses: {},           // { cellIdx: count }  (5 = отель)
  mortgaged: {},        // { cellIdx: true }   (залоги)
  logs: [],             // история транзакций
  hint: string,         // подсказка для игрока
  toast: string,        // всплывающее сообщение
}
```

## 📝 Главные функции для редактирования

### В `game-full.jsx`:

```javascript
function isMonopoly(region) { ... }         // проверка монополии
function buyProperty(pIdx, cellIdx) { ... } // покупка
function payRent(fromIdx, toId, amount) { ... } // оплата аренды
function rollDice() { ... }                 // бросок кубиков
```

## 🔗 Как события связаны друг с другом

```
rollDice() → stepMove() → land() → 
├─ buyable → phase='buy' → buy() → endTurn()
├─ rent → phase='rent' → payRent() → endTurn()
├─ event → handleEvent() → endTurn()
└─ special → phase='special' → endTurn()

endTurn() → nextPlayer() → phase='idle' (wait for rollDice)
```

## ❌ Частые ошибки

| Ошибка | Решение |
|--------|---------|
| "Cannot read property 'Board'" | Проверь, что `Board.jsx` загружен раньше `game-full.jsx` в HTML |
| Игрок застрял | Проверь `phase` в консоли, может быть неправильный переход |
| Кубики не бросаются | Убедись, что `phase === 'idle'` перед rollDice |
| Доска не рисуется | Проверь `owners` и `houses` структуру |
| Вкладки не работают | Убедись, что `window.useGame` экспортирован |
